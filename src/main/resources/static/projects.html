Generic Data Fetcher

This was my senior capstone team project during Spring 2022. Led by sponsor Professor Mario Kim with Openprise, we built a web application to convert various file types to the CSV format. I was primarily responsible for building microservices in the backend to download and convert the files using Groovy and the Micronaut framework as well as designing an object oriented database to store form inputs given from the form page to feed into our backend.

To find out more, check out the repo here!

Languages and Technology used:
  • Groovy
  • Microanut
  • Gradle
